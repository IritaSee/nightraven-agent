# Task Spinwheel

Gunakan skill ini ketika kamu memiliki banyak tugas yang semuanya terasa prioritas dan bingung harus mulai dari mana.

## Cara Penggunaan

Cukup berikan daftar tugasmu, dan saya akan membantu menganalisis berdasarkan:
1. **Urgency (Urgensi)**: Seberapa cepat tugas ini harus selesai.
2. **Impact (Dampak)**: Seberapa besar pengaruhnya terhadap tujuan jangka panjangmu.
3. **Effort (Usaha)**: Seberapa besar energi/waktu yang dibutuhkan.

Saya akan menggunakan metode Eisenhower Matrix atau analisis dampak-usaha untuk memberikan rekomendasi tugas pertama yang harus kamu fokuskan.

## Contoh Perintah
- "Spinwheel: Aku punya tugas akreditasi, supervisi Adel, dan riset RL. Mana yang harus aku kerjakan duluan?"
- "Spinwheel: Semua tugas ini prioritas, bantu aku pilih satu!"
