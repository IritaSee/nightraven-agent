# Persona-to-Narrative Pipeline

This skill defines the end-to-end pipeline for extracting behavioral, cognitive, and linguistic profiles from raw text or screenshots, and passing those profiles to a creative storytelling engine to generate high-fidelity character narratives.

## When to Use
Use this skill when the user wants to:
1. Analyze raw behavioral data (chat logs, social media screenshots, or text descriptions) to create a character profile.
2. Seamlessly feed that profile into a creative storytelling engine (like Honour Scribe) to produce immersive, psychologically rich narratives.
3. Maintain high fidelity and consistency between the analyzed persona and the resulting story.

## Pipeline Steps

### Step 1: Profile Extraction (Persona Profiler)
1. Use the `persona-profiler` skill or run `profile_generator.py` to analyze the input text or screenshots.
2. Extract key behavioral, cognitive, linguistic, and psychological patterns.
3. Generate a Markdown-formatted Persona Profile and an LLM-compatible Persona Engine system prompt.
4. **Important**: Deliver the Persona Profile directly as Markdown in the chat. Skip `.docx` file creation for this step.

### Step 2: Storytelling Engine Handover (Honour Scribe)
1. Feed the generated Persona Profile and Persona Engine prompt into the `honour-scribe` (Honour_Scribe_v2.0) engine.
2. Ensure the storytelling engine adopts the exact psychological tension, behavioral quirks, and linguistic style identified in Step 1.

### Step 3: Narrative Generation
1. Generate the creative story/narrative focusing on deep character profiling, psychological tension, and immersive, sensory-rich scenarios.
2. Deliver the final narrative in the requested format (e.g., direct chat or `.docx` if requested for academic/formal drafts).

## Output Format
The final output should clearly separate:
1. **The Persona Profile**: Delivered directly in chat as Markdown.
2. **The Narrative**: The immersive story generated by the storytelling engine based on the profile.

## Example Workflow

### Input
User provides a series of chat screenshots or a text description of a character's unique speaking style and behavioral habits.

### Execution
1. **Analyze & Profile**:
   - Run analysis on the input data.
   - Generate Markdown profile containing cognitive style, linguistic markers, and behavioral traits.
2. **Narrative Generation**:
   - Pass the profile to the Honour Scribe engine.
   - Generate a story where the character's actions and dialogue perfectly match the profiled traits.

### Output
```markdown
### Persona Profile
- **Linguistic Style**: Fast-paced, uses casual Indonesian slang mixed with professional English.
- **Cognitive Style**: High-stress, detail-oriented.

### Character Narrative
The rain drummed against the window pane as Andi paced the room. "Gue musti kelar hari ini," he muttered, his fingers flying across the keyboard with obsessive precision...
```
