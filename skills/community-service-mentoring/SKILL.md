# Community Service Mentoring Workflow

This skill provides a structured approach for community service (Pengabdian Masyarakat) projects that involve technical implementations (e.g., security systems, digital management). It emphasizes moving beyond "install-and-forget" by integrating digital literacy and mentoring to ensure long-term sustainability.

## When to Use
- When conducting community service projects involving technology deployment.
- When the goal is to empower local administrators (e.g., mosque takmir, village officials) to manage systems independently.
- When writing papers for community service journals (e.g., JARDIRA, Martabe, Jurdimas).

## Steps

### 1. Situational Analysis & Gap Identification
- Identify the "Gap": Contrast the common "install-and-forget" approach with the need for sustainable, literacy-driven systems.
- Assess current digital literacy levels of the target community.

### 2. Integrated Cluster Design
Organize the project into three core clusters:
- **Security/Technical**: The hardware/software setup (e.g., CCTV with NVR for scalability).
- **Governance**: Digital management protocols and administrative workflows.
- **Literacy**: Mentoring and training programs for the users.

### 3. Technical Implementation
- Select scalable and remote-capable hardware (e.g., NVR over DVR).
- Ensure the setup is documented for non-technical users.

### 4. Mentoring & Training (The Sustainability Phase)
- Conduct hands-on training sessions with administrators.
- Focus on troubleshooting, basic maintenance, and system utilization.
- Transition from "expert-led" to "administrator-managed" operations.

### 5. Evaluation & Reporting
- Measure the increase in digital literacy and system independence.
- Frame results in terms of "Sustainable Digital Transformation" for academic publication.

## Output Format
- **Project Plan**: Structured by the three clusters (Security, Governance, Literacy).
- **Manuscript Content**: Gap analysis focusing on sustainability and mentoring methodology.

## Example
**Project**: Smart Mosque Security System
- **Gap**: Previous attempts at security were "install-and-forget," leading to system failure when issues arose.
- **Security Cluster**: Installed 4MP HD CCTV with NVR for remote monitoring.
- **Governance Cluster**: Established a logbook for security checks.
- **Literacy Cluster**: Mentored the mosque takmir on how to access footage via mobile apps and perform basic NVR restarts.
- **Outcome**: Takmir can now manage the system without external technical support.
