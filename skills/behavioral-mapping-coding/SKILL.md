# Behavioral Mapping Coding

Use this skill to standardize data collection during behavioral mapping observations, ensuring consistency across student groups or research teams.

## When to Use
- Community psychology field observations.
- Public space usage studies.
- Ergonomics and environmental psychology research.

## Coding Symbols
- **L**: Laki-laki (Male)
- **P**: Perempuan (Female)
- **D**: Duduk (Sitting)
- **B**: Berdiri (Standing) - *Standard extension*
- **J**: Berjalan (Walking) - *Standard extension*

## Steps
1. **Define Observation Zone**: Map the physical boundaries of the area.
2. **Time Sampling**: Decide on observation intervals (e.g., every 15 minutes).
3. **Data Entry**: Mark the location of individuals on the map using the codes.
    - Example: `L-D` (Male, Sitting)
    - Example: `P-B` (Female, Standing)
4. **Aggregation**: Count totals per category to identify patterns of space usage.

## Output Format
- **Location**: [Name of space]
- **Time**: [Start - End]
- **Map/Grid**: [Reference to visual map]
- **Tally Table**:
    | Code | Count | Notes |
    |------|-------|-------|
    | L-D  | 5     | Mostly near the entrance |
    | P-D  | 3     | Near the garden |

## Example
> **Observation**: At 10:00 AM in the Banjar hall, identified 4 L-D (elderly men sitting) and 2 P-B (women standing near the kitchen area).
