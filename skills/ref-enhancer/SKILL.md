# Ref-Enhancer

## How to use
- "Refine this paragraph: [text]"
- "Find references for: [claim]"

## Workflow
1. Identify key claims in the input text.
2. Use search tools (arxiv-watcher, exa-web-search) to find relevant papers.
3. Synthesize findings and suggest citations (e.g., "Smith et al., 2023").
4. Suggest phrasing improvements for academic tone.
