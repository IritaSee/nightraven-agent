# KOL Brief Storytelling

Create Indonesian KOL (Key Opinion Leader) collaboration briefs that prioritize storytelling and clear audience segmentation.

## When to Use
- When preparing collaboration briefs for Indonesian influencers/KOLs.
- When the product has distinct target audiences (e.g., UKAI prep for pharmacy vs. Koas for medical).
- When the user wants a "humanized" or "storytelling" approach rather than a dry technical brief.

## Steps
1. **Identify Target Audience**: Define specific segments (e.g., Pharmacy students, Medical students, Professionals).
2. **Define Storytelling Hook**: Create a compelling "hook" (e.g., "The Mawapres Secret Sauce", "The 30-Second Life Saver").
3. **Structure the Brief**:
    - **Campaign Overview**: Brief context of the product/project.
    - **Target Audience**: Detailed section for each segment.
    - **Content Pillars**: Story-driven themes (e.g., "The Aesthetic Clinical Toolkit").
    - **Key Messages**: Core points to convey in Indonesian.
    - **Deliverables**: Specific requirements (Reels, Stories, etc.).
4. **Format**: Use Times New Roman 11pt, 1.15 line spacing.
5. **Delivery**: Save as `.docx` and deliver via Telegram if requested.

## Output Format
- A structured document in Indonesian.
- Clear headings for "Target Audience" and "Content Pillars".
- Storytelling-focused language.

## Example
**Hook**: "The Mawapres Secret Sauce"
**Target Audience**: Mahasiswa Farmasi (Persiapan UKAI) & Apoteker Muda.
**Content Pillar**: "The 30-Second Life Saver" - Menunjukkan bagaimana SakuMedis membantu dalam situasi klinis kritis dengan cepat.
